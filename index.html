Infraestrutura como Código (IaC)

Automação de provisionamento em nuvem com Terraform e Ansible

Este projeto apresenta uma abordagem de Engenharia de Software aplicada ao DevOps, utilizando Infraestrutura como Código para automação, escalabilidade e padronização de ambientes cloud.

Objetivo do Projeto

Este projeto propõe a automação do provisionamento e da configuração de infraestrutura em ambientes de nuvem por meio das ferramentas Terraform e Ansible, adotando práticas de Infrastructure as Code (IaC) e princípios da cultura DevOps. A solução visa garantir a criação declarativa, versionada e reprodutível de recursos de infraestrutura, reduzindo intervenções manuais e minimizando inconsistências entre ambientes.
